# 💼 Portfolio Personal

Faqe personale portfolio per te shfaqur projektet dhe aftesine si developer.

## ✨ Seksionet
- Hero me prezantim personal
- Rreth meje + aftesitë teknike
- Galeria e projekteve
- Seksioni i kontaktit

## 🚀 Si ta perdoresh
1. Hap `index.html` ne browser
2. Personalizoni tekstin, emrin dhe emailin tuaj
3. Shto linket e projekteve reale

## 🛠️ Teknologjite
- HTML5 (Semantike)
- CSS3 (Glassmorphism, CSS Variables, Flexbox/Grid)
- JavaScript minimal
